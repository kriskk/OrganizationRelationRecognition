Zend Server — полнофункциональный web-стек, поставляемый в одном пакете. Включает в себя http-сервер Apache, PHP, Zend Framework, набор модулей для поддержки БД (Oracle, DB2, MySQL, PostgreSQL), коннектор к Java, специально разработанный web-интерфейс для управления конфигурацией, отладчик, оптимизатор Байткода и API для организации кеширования.
