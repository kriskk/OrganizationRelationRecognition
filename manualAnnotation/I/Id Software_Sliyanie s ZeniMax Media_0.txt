Слияние с ZeniMax Media

24 июня 2009 года было анонсировано о том, что компания ZeniMax Media, которая является родительской компанией Bethesda Softworks, завершила приобретение id Software. Согласно договору, Bethesda Softworks будет издавать все игры id Software за исключением тех, которые уже имеют соглашение на издание с другими компаниями.
