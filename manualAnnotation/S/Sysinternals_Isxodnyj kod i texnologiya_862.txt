Некоторые приемы, которые используются для кодирования информации, были основаны на Windows Native API (NTAPI), они все ещё являются слабо документированными корпорацией Microsoft. Используя примеры этих кодировок, разработчики могут создавать чрезвычайные программы, которые могут производить такие операции, которые не под силу выполнить стандартными API.
