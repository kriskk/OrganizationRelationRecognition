Для удаленного управления виртуальными машинами, запущенными в среде Parallels Desktop, с помощью мобильных устройств Apple, существует программа Parallels Mobile.
 Parallels Server для Mac — первая система виртуализации серверов Apple на базе Процессора Intel, включая Xserve и Mac Pro, поддерживающая до 50-ти различных гостевых операционных систем (в том числе 32- и 64-разрядные версии Windows Server и множество версий Linux). Кроме того, это первая в мире система, поддерживающая Mac OS X Leopard Server в качестве гостевой операционной системы виртуальной машины. Выпуск этого решения позволил устранить препятствия на пути массового использования инфраструктуры Apple в современных дата-центрах. Существует версия Parallels Server для Mac Mini, которая отличается от Parallels Server для Mac исключительно наличием ограничения на максимум две одновременно запущенные виртуальные машины.
