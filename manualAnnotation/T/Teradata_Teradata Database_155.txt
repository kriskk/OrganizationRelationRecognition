 СУБД Teradata Database — это линейная и предсказуемая во всех направлениях нагрузка на базу данных (тома данных, пропускная способность, число пользователей, сложность запросов). Масштабируемость объясняется популярностью приложений для хранения данных, используемых в промышленных масштабах. Teradata предлагает серверы Intel, соединённые в частную сеть BYNET для обмена сообщениями. Системы Teradata предлагаются с фирменными Teradata’вскими дисковыми массивами для хранения баз данных производства либо LSI, либо EMC.
